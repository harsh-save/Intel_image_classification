# Intel Image Classification

#### This ML model tries to classify the images in 6 classes.

#### The Classes and their mappings are are:

1. 'buildings' -> 0
2. 'forest' -> 1
3. 'glacier' -> 2
4. 'mountain' -> 3
5. 'sea' -> 4
6. 'street' -> 5

#### The project uses Tensorflow as the Deep Learning backend.
